Knappar

Den här sidan hänger ihop med hero sidan i ganska så stor utsträckning. Det för att samtliga heroes på den sidan använder sig av knappar.

Vad

En knapp på en webbsida är inte alltid en knapp, utan ofta är det faktiskt länkar som är stylade som knappar. Detta är för att oftast så leder knappen och knapptrycket till en sida och då ska ett länk element användas för att det funktionellt och semantiskt ska vara korrekt. Knappar är oftast att något ska skickas med ett formulär eller liknande, du kan kika på type egenskapens värden på MDN för att se detta.

Hur

För att skapa en knapp så använder vi oss av antingen ett a-element eller ett button-element. Det är viktigt att tänka på att använda rätt element för rätt syfte. Om det är en länk så ska det vara ett a-element, om det är en knapp så ska det vara ett button-element.

Grunden

Vi börjar med att skapa en knapp. Vi använder oss av ett a-element och stylar det som en knapp. Vi använder display: inline-block; för att kunna sätta en bredd och höjd på elementet. Vi använder också padding för att skapa lite utrymme runt texten i knappen.

Button länk 

  .button {
    display: inline-block;
    padding: 1rem 2rem;
    background-color: #333;
    color: #fff;
    text-decoration: none;
  }

Storlek, höjd

För att skapa en knapp som är större så kan vi använda oss av padding. Vi kan också använda oss av font-size för att göra texten större. Vi kan också använda oss av border-radius för att skapa rundade hörn på knappen.


  .button--large {
    padding: 2rem 4rem;
    font-size: 2rem;
    border-radius: 1rem;
  }

Hover

För att skapa en hover-effekt på knappen så kan vi använda oss av :hover pseudo-klassen. Vi kan ändra bakgrundsfärgen och textfärgen för att skapa en effekt när användaren hovrar över knappen.


  .button:hover {
    background-color: #555;
    color: #eee;
  }

Active

För att skapa en active-effekt på knappen så kan vi använda oss av :active pseudo-klassen. Vi kan ändra bakgrundsfärgen och textfärgen för att skapa en effekt när användaren klickar på knappen.


  .button:active {
    background-color: #777;
    color: #ddd;
  }
