CSS variabler, eller "custom properties"

Att kunna använda variabler i CSS är otroligt praktiskt när det kommer till att kunna ändra saker snabbt och relativt enkelt. Det låter dig till exempel att deklarera färger på ett ställe i din css och du kan sedan använda dem om och om igen utan att behöva skriva en hex kod eller rgb värden. Förutom det så blir det väldigt enkelt att justera färger och typsnitt på många olika ställen i din design, utan att behöva söka igenom en massa regler.

Hur?

Så hur fungerar css variabler? Som i så många andra språk så behöver du deklarera variabeln, alltså skapa den och tilldela den ett värde. Syntaxet för en css variabel börjar med -- 2 stycken sträck följt av variabelns namn. För att använda den sedan så används var() funktionen.

Variabeln eller egenskapen behöver skrivas inuti en css-regel som alla andra css-egenskaper.

    
      .regel {
        --egenskap: värde;
        width: var(--egenskap);
      }

På det här sättet kan vi skapa och använda egenskaper i vår css. Men viktigt i det här fallet är att --egenskap är lokal till vår css-regel, det vill säga att den inte går att använda utanför .regel. För att göra en variabel global och kunna återanvända den så kan vi använda pseudo-klassen :root.

För att deklarera variabler med root så kan vi göra det i början på css dokumentet.

    
      :root {
        --egenskap: värde;
      }

      .regel {
        width: var(--egenskap);
      }

Ett exempel

När vi nu vet hur css-variabler fungerar så kan vi använda dem till att deklarera färger, typsnitt och mått som vi kommer använda i vår design.

I det här exemplet så hittar du ett antal vanliga egenskaper som används på webbsidor.


  :root {
    --color-dark: #333;
    --color-light: #fff;
    --color-primary: #007bff;
    --color-secondary: #6c757d;
  
    --container-max: 60rem;
    --gutter: 2rem;
  
    --font-size: 1.2rem;
    --line-height: 1.5;
    --font-body: sans-serif;
    --font-heading: serif;
  }

När variablerna är på plats så använder du dem senare i dina css regler.


  body {
    font-family: var(--font-body);
    font-size: var(--font-size);
    line-height: var(--line-height);
    color: var(--color-dark);
    background-color: var(--color-light);
  }
  h1, h2, h3 {
    font-family: var(--font-heading);
    line-height: 1;
  }
  .container {
    width: min(100% - var(--gutter), var(--container-max, 60ch));
    margin-inline: auto;
  }
