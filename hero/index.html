Hero - produkthero

Vad och varför

En hero på en sida är en form av header. Det är något som användaren möts av när de besöker din sida och det är din chans att göra ett intryck som varar.

Ofta så innehåller en hero en så kallad "call to action", det vill säga något som användaren önskas utföra.

Det kan vara att köpa en produkt, att skapa ett konto eller att prenumerera på ett nyhetsbrev.

Hur

För att skapa en hero så kommer vi starta med en grund i css. Vi kommer använd display: grid; för att kunna placera element ovanpå varandra. Men vi vill även kunna styra var elementen landar och i vilken ordning.

Grunden

Vi börjar med att skapa en container för vår hero. Vi använder display: grid; och grid-template-areas för att kunna styra var elementen hamnar.


  .hero {
    display: grid;
    grid-template-areas: "hero";
    align-items: center;
  }

Storlek, höjd

Först så kommer vi att styra höjden på vår hero, bredden förutsätter vi är hela skärmen. Vi använder måttet vh, viewport height för att kunna sätta en höjd baserade på hur stor användarens webbläsarfönster är. Vi sätter också max-height för att heron inte ska täcka flera skärmar på mindre devices.


  :root {
    --hero-height: 60vh;
    --hero-max-height: 500px;
  }
  .hero {
    height: var(--hero-height);
    max-height: var(--hero-max-height);
  }

Bilden

Nästa del är att styra bildens storlek. Bilden ligger i elementet med .hero klassen så vi kan komma åt den med en css-selektor som väljer img element i .hero.


  .hero > img {
    object-fit: cover;
    height: min(var(--hero-height), var(--hero-max-height));
    width: 55%;
    
    justify-self: end;
  }

Så vad gör detta, object-fit: cover; gör så att bilden täcker hela elementet. Height styr storleken och width bredden på elementet.

För att bilden ska placeras till höger om texten så använder vi justify-self: end; Här kan du såklart prova annan placering beronde på var din text finns.

Texten

Slutligen så justerar vi texten för att få en marginal och lägga den på sidan om bilden.


  .hero-content {
    margin-left: 5%;
    max-width: 40%;
    min-width: 30ch;
  }

Allt detta är exempel och det behöver justeras utifrån din bild och din text.

Små skärmar

På små skärmar så kommer en hero nästa alltid få problem. Då kan det vara aktuellt att skapa undantag för stilarna med hjälp av media regler.

För att skapa ett undantag så skriver du i din img regel något liknande detta, du skapar då kod som bara är aktuell när villkoret för regeln är sann (min-width: 60rem).


.hero > img {
  ...
  @media (min-width: 60rem) {
    width: 60%;
  }
}

Hero - text på bild heron

Vad och varför

Precis som tidigare så är det samma gällande budskapet, men vi gör det på annat sätt.

Hur

Mycket enklare än den första delen. Bilden får täcka hela elementet och vi styr ordningen på elementen med z-index. Tänk på elementen som lager som vi kan placera display:grid för att placera allt innehåll i mitten.

För att förbättra kontrasten så sätter vi sedan bakgrundsfärgen. Bakgrundsfärgen är transparent och förmörkar på så sätt bilden, vilket förbättrar kontrasten för den vita texten.

Hero med två kolumner

För det här sista exemplet så kräver det att vi ändrar på HTML innehållet.

Hur

HTML koden ändras så att den div som har klassen hero-content nu finns innuti en hero-wrapper. Wrapperns funktion är att omsluta de två kolumnerna och skapa dem med hjälp av grid.


.hero3-wrapper {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(30ch, auto));
  gap: 2rem
}

Detta placerar innehållet i 2 kolumner som även anpassas efter skärmens storlek.

Med det på plats så kan vi ändra innehållets position med align-items och justify-content, precis som vi gjort tidigare med både heros och när vi arbetat med flex.

För att färdigställa exemplet så lägger vi till en padding för att få luft runt elementen. För det används måttet vmin vilket anpassar sig efter webbläsarfönstrets storlek.


.hero3-wrapper {
  ...
  padding: 10vmin 2rem;
}
